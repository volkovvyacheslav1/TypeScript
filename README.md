# TypeScript
В кадом из представленных заданий, после выполнения команд нужно поменять папку src на исходную.
Для начала работы над проектом и установки нужных зависимостей выполните в терминале следующие команды: npm init -y и npm install parcel cropperjs@1.5.13.
Чтобы запустить проект первого задания, используйте команду: npm run start.
Для реализации с третьего по шестое заданий процесс установки будет схожим: выполните команды npm create vite@latest todo-app -- --template vanilla-ts, затем cd todo-app и npm install.
Для запуска проектов с третьего по шестое заданий используйте: npm run dev.
Теперь перейдем к заданию семь: выполните команды cd 7, затем cd booking-project и npx jest.
Чтобы перемещаться между папками в терминале, используйте: cd .. — для возврата на уровень выше, cd <название папки> (вводите без угловых скобок) — чтобы войти в указанную папку.
Перед запуском проекта убедитесь, что находитесь в директории проекта, где находятся все необходимые файлы, включая package.json.
